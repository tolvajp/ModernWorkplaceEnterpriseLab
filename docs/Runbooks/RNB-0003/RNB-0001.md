# Runbook – Licensing baseline activation (Microsoft 365 E5)

## Status
Executed

---

## Objective
- Activate Microsoft 365 E5 licensing to enable enterprise-grade identity, endpoint, and security capabilities.
- Establish licensing entitlement without performing configuration or enforcement.

---

## Scope

### In Scope
- Activation of Microsoft 365 E5 (trial or subscription)
- Verification of license availability in the tenant
- Confirmation of entitlement for identity and endpoint services

### Out of Scope
- User creation
- License assignment to users
- Any security, identity, or endpoint configuration
- Policy creation or enforcement

---

## Preconditions
- [x] Related decision (DEC-0001) is in status **Accepted**
- [x] Tenant successfully created (RNB-0000 executed)
- [x] Global Administrator access available
- [x] No conflicting licensing runbook executed

---

## Risks & Safety Notes
- Trial-based licenses have time limitations
- License activation may expose additional configuration options
- Some services may prompt for configuration after activation

Mitigations:
- No configuration changes performed in this runbook
- Feature usage deferred to subsequent decisions and runbooks

---

## Procedure

### Step 1 – Preparation
- Confirm tenant access and administrative permissions
- Review licensing scope defined in DEC-0001
- Ensure no users or devices are yet dependent on licensing

---

### Step 2 – Execution
- Activate Microsoft 365 E5 licensing (trial or subscription) for the tenant
- Accept required terms and conditions
- Do not assign licenses to users
- Do not configure any services beyond license activation

*(Screenshots may be added for major navigation context if required)*

---

### Step 3 – Validation
- Verify Microsoft 365 E5 licenses are available in the tenant
- Confirm availability of:
  - Microsoft Entra ID P2 features
  - Microsoft Intune entitlement
- Confirm no licenses are assigned to users

---

## Validation Checklist
- [x] Licensing activated successfully
- [x] No licenses assigned to users
- [x] No out-of-scope configuration performed
- [x] Tenant remains in a pre-configuration state

---

## Rollback / Exit Strategy
- Remove licenses or cancel subscription if applicable

---

## Notes
This runbook intentionally enables entitlement only.
All functional use of licensed features is handled in subsequent decisions and runbooks.
